Projets

To Do List

To-List App est une application Web simple développée en Angular qui vous permet de créer, afficher, modifier et supprimer des éléments d'une liste. Cette application utilise des concepts angulaires fondamentaux, notamment la liaison de données et les directives structurelles.

Crud

Cette application est un exemple de système CRUD (Create, Read, Update, Delete) développé avec Angular. Il permet aux utilisateurs d'afficher, de créer, de modifier et de supprimer des éléments à partir d'une interface utilisateur intuitive.

Trello

Trello App est une application web simple développée en Angular qui permet d'ajouter, de visualiser, d'éditer et de supprimer des tâches.Cette application utilise des concepts fondamentaux d'Angular, notamment la liaison de données et les directives structurelles.

Weather App

Bienvenue dans le projet Weather App, une application simple développée avec Angular.Cette application vous permet de visualiser les températures des villes par liste ou individuellement. Les statistiques des villes sont également enregistrées.

Download-Upload

Bienvenue dans le projet Download-Upload Ce projet, créé avec Angular, permet aux utilisateurs d'importer et de télécharger des fichiers de type image, PDF, et autres. Il offre une interface simple et intuitive pour gérer les opérations de téléchargement et d'importation de fichiers.
